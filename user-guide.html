Tez-tez so'raladigan savollar

Tizimni qanday ishga tushiraman?

Development muhiti uchun composer run dev buyrug'ini ishlating. Bu server, queue, logs va vite'ni bir vaqtda ishga tushiradi.

Testlarni qanday ishlataman?

Barcha testlar uchun: composer run test
Alohida test uchun: php artisan test --filter=testName
Pest v4 browser testing ham mavjud.

Kodni qanday formatlaydi?

PHP uchun: vendor/bin/pint --dirty
JavaScript uchun: npm run format
Commit qilishdan oldin albatta formatlaing.

Mavzuni (theme) qanday o'zgartiraman?

Header'dagi mavzu tugmasini bosing yoki Sozlamalar > Appearance sahifasiga o'ting. Light, Dark va System variantlari mavjud.
